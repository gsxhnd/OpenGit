# 1. 项目概述

## 1.1 定位

OpenGit 是一个使用 Rust 和 GPUI 框架构建的**跨平台 Git GUI 客户端**。它的核心差异化特性是在**单一窗口中管理多个 Git 仓库**，面向需要同时维护多个项目的开发者。

## 1.2 核心理念

| 理念 | 说明 |
|------|------|
| **高性能** | Rust 原生编译，GPUI GPU 加速渲染，大仓库也流畅 |
| **多项目管理** | 侧边栏项目列表 + Tab 切换，一个窗口管理所有仓库 |
| **现代化 UI** | 基于 Zed 编辑器的 GPUI 框架，支持 21+ 主题 |
| **功能完整** | 覆盖日常 Git 操作，并提供 Graph 可视化、AI Commit、平台集成等高级功能 |
| **跨平台** | 支持 Linux (X11/Wayland)、macOS、Windows |

## 1.3 目标用户

- 需要同时管理多个 Git 仓库的全栈开发者
- 偏好 GUI 操作但不满意现有工具性能的开发者
- 希望获得 Git Graph 可视化和 AI 辅助 commit 的开发者
