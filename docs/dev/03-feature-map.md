# 3. 功能地图

> 标记说明：✅ 已实现 | 🚧 开发中 | 📋 计划中 | 🔮 远期

## 3.1 功能优先级

| 优先级 | 功能 | 目标 |
|--------|------|------|
| P0 | UI 工作台重构 | 建立 Activity Bar、Sidebar、Session Tabs、Status Bar 框架 |
| P0 | SSH + 本地终端 | 统一终端体验，支持远程 Shell 与本地 Shell |
| P0 | SFTP 文件管理 | 远程文件浏览、上传下载、文本编辑、传输反馈 |
| P1 | FTP/WebDAV/S3 | 将传输、编辑能力抽象到统一文件能力模型 |
| P2 | VNC/RDP | 远程桌面会话，复用连接中心与会话工作区 |
| P2 | Docker/K8s | 容器/集群管理，复用连接与会话模型 |

## 3.2 SSH 远程访问

| 功能 | 说明 | 状态 |
|------|------|------|
| SSH 连接与会话 | 快速连接、已保存主机连接、会话 Tab 管理 | 🚧 |
| SSH 远程终端 | xterm.js 渲染远程 Shell 输出 | 🚧 |
| 多主机管理 | 主机列表增删改查、分组/标签、连接状态概览 | 📋 |
| 多种认证 | 密码、私钥（PEM/OpenSSH）、SSH Agent、keyboard-interactive | 📋 |
| 端口转发 | 本地转发（-L）、远程转发（-R）、动态转发/SOCKS5（-D） | 📋 |
| 主机指纹 | 指纹识别、信任存储、首次连接确认 | 📋 |
| 远程资源监控 | SSH 会话内实时展示 CPU、内存、磁盘使用率 | 📋 |
