# 3. 功能规划

> 标记说明: ✅ 已实现 | 🚧 开发中 | 📋 计划中

## 3.0 远程访问与云存储（产品方向）

| 功能 | 说明 | 状态 |
|------|------|------|
| **SSH 会话 / 终端** | 安全 Shell、多会话与凭据管理；终端 UI 采用 **xterm.js**（`@xterm/xterm`） | 📋 |
| **SFTP 文件管理** | 浏览、上传、下载、权限与断点续传等 | 📋 |
| **远程文档编辑** | 对远程文本文件使用 **Monaco Editor** 打开与保存（语法高亮、主题与编辑器能力对齐本地 IDE 体验） | 📋 |
| **Docker** | 本机或已连接环境上的容器/镜像视图、日志、exec、启停等（具体范围随里程碑定义；依赖 Docker CLI 或 API） | 📋 |
| **Kubernetes** | 多集群/上下文、命名空间与 Workload 浏览、Pod 日志、端口转发等（以 `kubectl` 兼容工作流为主，能力逐步扩展） | 📋 |
| **WebDAV** | 挂载与文件同步；**里程碑排序在 Docker/K8s 之后** | 📋 |
| **S3 兼容 API** | 对象存储（含 MinIO、Cloudflare R2 等兼容端点） | 📋 |

**支持程度与路线顺序**：Docker 与 Kubernetes 列为**正式规划能力**，整体交付节奏与能力深度**优先于 WebDAV**。细则见 [09-roadmap.md](09-roadmap.md) Phase 10–13。

## 3.2 快捷键（规划）

默认方案以**远程连接、会话切换、传输队列**为主（具体键位随实现迭代）：

| 快捷键 | 操作（示意） |
|--------|----------------|
| `Ctrl/Cmd + Shift + P` | 命令面板 |
| `Ctrl/Cmd + ,` | 打开设置 |
| 其他 | 新建会话、聚焦终端、打开传输队列等（定稿时补充） |
