# 3. 功能规划

> 标记说明: ✅ 已实现 | 🚧 开发中 | 📋 计划中

## 3.0 远程访问与云存储（产品方向）

| 功能 | 说明 | 状态 |
|------|------|------|
| **SSH 会话 / Tab 管理** | 每个 SSH 连接成功后创建独立 Tab 页；多连接即多 Tab，支持切换与关闭；Tab 内含三个子视图（终端、SFTP、资源监控） | 🚧 |
| **SSH 终端子视图** | Tab 内终端页面，采用 **xterm.js**（`@xterm/xterm`）渲染远程 Shell 输出 | 🚧 |
| **SFTP 双栏对比子视图** | Tab 内 SFTP 页面，左栏显示**本地文件系统**，右栏显示**远程服务器文件系统**，支持双向拖拽传输 | 📋 |
| **远程资源监控子视图** | Tab 内监控页面，实时展示远程服务器 CPU 使用率、内存占用、磁盘使用率等指标（通过 SSH 执行命令采集） | 📋 |
| **多主机管理** | 主机列表统一管理（增删改查）、分组/标签、快速切换、连接状态概览 | 📋 |
| **SSH 多种登录认证** | 密码、私钥（PEM/OpenSSH）、SSH Agent 转发、键盘交互（keyboard-interactive）、证书认证 | 📋 |
| **SSH 端口转发** | 本地转发（-L）、远程转发（-R）、动态转发/SOCKS5 代理（-D）；转发规则持久化与状态监控 | 📋 |
| **SFTP 文件管理（高级）** | 目录树、权限显示、断点续传、传输队列、覆盖策略、进度反馈 | 🚧 |
| **远程文档编辑** | 对远程文本文件使用 **Monaco Editor** 打开与保存（语法高亮、主题与编辑器能力对齐本地 IDE 体验） | 🚧 |
| **Docker** | 本机或已连接环境上的容器/镜像视图、日志、exec、启停等（具体范围随里程碑定义；依赖 Docker CLI 或 API） | 📋 |
| **Kubernetes** | 多集群/上下文、命名空间与 Workload 浏览、Pod 日志、端口转发等（以 `kubectl` 兼容工作流为主，能力逐步扩展） | 📋 |
| **WebDAV** | 挂载与文件同步；**里程碑排序在 Docker/K8s 之后** | 📋 |
| **S3 兼容 API** | 对象存储（含 MinIO、Cloudflare R2 等兼容端点） | 📋 |

> **产品定位说明**：Puck **不提供本地终端**，所有终端与文件操作均面向远程服务器。本地文件系统仅在 SFTP 双栏对比视图中作为传输源/目标使用。

**支持程度与路线顺序**：Docker 与 Kubernetes 列为**正式规划能力**，整体交付节奏与能力深度**优先于 WebDAV**。细则见 [09-roadmap.md](09-roadmap.md)：**Phase 2**（SSH/SFTP/Monaco，含 Tab 管理、双栏 SFTP、资源监控、多主机管理、多种认证、端口转发）、**Phase 3**（打包发布）、**Phase 4–6**（Docker/K8s、WebDAV、S3）。

## 3.2 快捷键（规划）

默认方案以**远程连接、会话切换、传输队列**为主（具体键位随实现迭代）：

| 快捷键 | 操作（示意） |
|--------|----------------|
| `Ctrl/Cmd + Shift + P` | 命令面板 |
| `Ctrl/Cmd + ,` | 打开设置 |
| 其他 | 新建会话、聚焦终端、打开传输队列等（定稿时补充） |
