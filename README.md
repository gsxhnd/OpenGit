# Puck

Cross-platform remote workbench built with Electron, React, and TypeScript.

SSH remote access, local terminal, FTP/SFTP file transfer, remote text editing, and resource monitoring. WebDAV/S3 storage, VNC/RDP remote desktop, and Docker/Kubernetes are on the roadmap.

## Development

```bash
npm install
npm run dev          # Dev server + Electron hot reload
npm run build        # Production build
npm run typecheck    # Type check all processes
```
